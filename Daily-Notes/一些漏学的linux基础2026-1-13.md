## 命令别名 alias
定义别名NAME，其相当于执行命令VALUE  
alias NAME='VALUE  
取消定义别明   
unalias NAME='VALUE'  
如果别名同原命令同名，如果要执行原命令，可使用  
\ALIASNAME

## bash快捷键
| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl + L` | 清屏 |
| `Ctrl + ←` | 光标快速向左移动一个单词 |
| `Ctrl + U` | 删除从光标到行首的所有字符 |
| `Ctrl + K` | 删除从光标到行尾的所有字符 |
| `Ctrl + D` | 删除光标后的一个字符 |
